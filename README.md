An app template for large Flask applications with Flask-SQLAlchemy integration. The application has the following structure:

pip install -r requirements.txt (no codespace não precisa)

flask run --debug

```
.
├── app
│   ├── extensions.py
│   ├── __init__.py
│   ├── models
│   │   ├── post.py
│   │   └── question.py
│   ├── posts
│   │   ├── __init__.py
│   │   └── routes.py
│   └── questions
│   │   ├── __init__.py
│   │   └── routes.py
├── config.py
```
